## 2. 数据解析（Raw → CSV）

### 2.1 标注解析方法

- 使用 Python 解析 XML 文件
- 提取字段：filename、class、xmin、ymin、xmax、ymax
- 汇总生成 `raw_annotation_summary.csv`

---

## 3. 数据清洗与处理

### 3.1 清洗策略

- 删除无法读取的图片
- 删除无标注图片
- 删除异常标注：
  - bbox宽或高 < **5像素**
  - bbox越界
  - bbox面积为0
- 删除类别异常数据

---

### 3.2 正方形化处理

方法：
- 使用 padding 转为正方形

标注同步：
- bbox坐标随 padding 偏移更新

---

### 3.3 输出格式选择

选择：YOLO txt

理由：
- 与Ultralytics训练直接兼容
- 简单高效

---

### 3.4 数据划分

- 划分比例：7 / 2 / 1
- 随机种子：42
- 方法：随机划分

## 4. 数据增强

### 4.1 增强方法

- 水平翻转
- 亮度/对比度调整
- 随机缩放
- 随机裁剪
- 轻微旋转

### 4.2 标注同步

- 所有几何变换同步更新 bbox
- 使用相对坐标保证正确性

- 每张图至少1个标注 ✔
- 无越界框 ✔
- 无负面积 ✔
